# A008 月結報表收件信箱設定

## 功能說明
系統管理員可設定月結報表發送時，管理員側的收件 Email 清單。A005 月結報表執行時，除發送給廠商外，亦會發送至此清單中的所有信箱。

## 使用者角色
- 系統管理員

## 功能需求

### 系統管理員
- 查詢目前設定的報表收件信箱清單
- 新增報表收件信箱
- 刪除報表收件信箱

## API 路徑

| 方法 | 路徑 | 說明 |
|---|---|---|
| GET | /api/settings/report-emails | 查詢報表收件信箱清單 |
| POST | /api/settings/report-emails | 新增報表收件信箱 |
| DELETE | /api/settings/report-emails/{id} | 刪除報表收件信箱 |

## 流程說明
1. 管理員進入系統設定頁面
2. 查看目前報表收件信箱清單
3. 新增信箱：輸入 Email 後儲存，系統驗證格式並寫入資料庫
4. 刪除信箱：選擇欲刪除的 Email，系統從資料庫移除

## 資料表

### report_recipient_emails（報表收件信箱資料表）
- id（PK）
- email（收件信箱）
- created_by（FK → employees.id，建立者）
- created_at（建立時間）
