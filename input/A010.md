# A010 部門管理

## 功能說明
系統管理員可查詢、新增、修改部門。員工資料需關聯所屬部門。

## 使用者角色
- 系統管理員

## 功能需求

### 系統管理員
- 查詢部門清單
- 新增部門
- 修改部門名稱

## API 路徑

| 方法 | 路徑 | 說明 |
|---|---|---|
| GET | /api/admin/departments | 查詢部門清單 |
| POST | /api/admin/departments | 新增部門 |
| PATCH | /api/admin/departments/{id} | 修改部門名稱 |

## 流程說明

### 查詢部門清單
1. 管理員進入部門管理頁面
2. 系統回傳所有部門清單

### 新增部門
1. 管理員輸入部門名稱
2. 系統檢查部門名稱是否重複
3. 重複則顯示錯誤提示，不允許新增
4. 不重複則儲存新部門

### 修改部門名稱
1. 管理員選擇部門並修改名稱
2. 系統儲存變更

## 資料表

### department（部門資料表）
- id（PK）
- name（部門名稱）
- created_at（建立時間）

### employees（新增欄位）
- department_id（FK → department.id）
