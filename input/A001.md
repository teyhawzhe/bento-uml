# A001 內部員工登入驗證

## 功能說明
提供公司內部員工透過帳號密碼進行系統登入，並驗證身份後取得使用權限。

## 使用者角色
- 員工
- 系統管理員

## 功能需求

### 員工
- 輸入帳號與密碼進行登入
- 登入失敗時查看錯誤訊息
- 登出系統

### 系統管理員
- 新增員工帳號
- 停用員工帳號
- 重設員工密碼

## API 路徑

| 方法 | 路徑 | 說明 |
|---|---|---|
| POST | /api/auth/login | 員工登入 |
| POST | /api/auth/logout | 員工登出 |

## 流程說明
1. 員工於登入頁面輸入帳號與密碼
2. 系統驗證帳號是否存在
3. 帳號不存在則顯示錯誤訊息，流程結束
4. 帳號存在則驗證密碼是否正確
5. 密碼錯誤則顯示錯誤訊息，流程結束
6. 密碼正確則檢查帳號是否啟用中
7. 帳號已停用則顯示停用訊息，流程結束
8. 帳號正常則產生 JWT Token，回傳給前端，導向系統首頁

## 資料表

### employees（員工資料表）
- id（PK）
- username（帳號）
- password_hash（密碼雜湊）
- name（姓名）
- is_admin（是否為管理員）
- is_active（是否啟用）
- created_at（建立時間）
- updated_at（更新時間）
