# A007 管理員管理供應商

## 功能說明
管理員可查詢供應商清單，並修改供應商資料。供應商的 id 與 business_registration_no 不可修改，其餘欄位皆可更新。

> 供應商新增（POST /api/suppliers）定義於 A002。

## 使用者角色
- 系統管理員

## 功能需求

### 系統管理員
- 查詢供應商清單（支援依名稱精確查詢與模糊查詢）
- 查詢單一供應商詳細資料
- 修改供應商資料（name、email、phone、contact_person、is_active）

## 限制
- id 與 business_registration_no 不可修改

## API 路徑

| 方法 | 路徑 | 說明 |
|---|---|---|
| GET | /api/suppliers | 查詢供應商清單（可帶 name、search_type 參數，search_type 為 exact 或 fuzzy） |
| GET | /api/suppliers/{id} | 查詢單一供應商詳細資料 |
| PATCH | /api/suppliers/{id} | 修改供應商資料 |

## 流程說明

### 查詢供應商清單
1. 管理員進入供應商管理頁面
2. 可輸入名稱並選擇查詢方式：
   - 精確查詢：名稱完全符合（search_type=exact）
   - 模糊查詢：名稱包含關鍵字（search_type=fuzzy）
3. 系統依查詢方式回傳符合條件的供應商清單

### 修改供應商資料
1. 管理員從清單選取欲修改的供應商
2. 系統顯示該供應商現有資料（id 與 business_registration_no 顯示但不可編輯）
3. 管理員修改可編輯欄位並送出
4. 系統驗證欄位格式（如 email 格式）
5. 驗證失敗則顯示錯誤訊息，不儲存
6. 驗證通過則更新資料庫並顯示修改成功

## 資料表

### suppliers（供應商資料表，參照 A002）
- id（PK，不可修改）
- name（供應商名稱，可修改）
- email（通知信箱，可修改）
- phone（聯繫電話，可修改）
- contact_person（負責人，可修改）
- business_registration_no（營業登記編號，不可修改）
- is_active（是否啟用，可修改）
- created_at（建立時間）
