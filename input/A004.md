# A004 系統錯誤通知信箱設定

## 功能說明
系統管理員可設定當系統發生錯誤時的通知 Email 信箱。

## 使用者角色
- 系統管理員

## 功能需求

### 系統管理員
- 新增錯誤通知信箱
- 刪除錯誤通知信箱
- 查詢目前設定的錯誤通知信箱清單

## API 路徑

| 方法 | 路徑 | 說明 |
|---|---|---|
| GET | /api/settings/error-emails | 查詢錯誤通知信箱清單 |
| POST | /api/settings/error-emails | 新增錯誤通知信箱 |
| DELETE | /api/settings/error-emails/{id} | 刪除錯誤通知信箱 |

## 流程說明
1. 管理員進入系統設定頁面
2. 查看目前錯誤通知信箱清單
3. 新增信箱：輸入 Email 後儲存，系統驗證格式並寫入資料庫
4. 刪除信箱：選擇欲刪除的 Email，系統從資料庫移除

## 資料表

### error_notification_emails（錯誤通知信箱資料表）
- id（PK）
- email（通知信箱）
- created_by（FK → employees.id，建立者）
- created_at（建立時間）
