# A015 便當菜單設定提醒

## 使用者角色
- 管理員
- 系統

## 主要流程
1. 系統每日排程（早上 8:00）檢查後一個月內的上班日是否有未設定菜單的日期
2. 若有未設定日期，發送 Email 至系統錯誤通知信箱，列出未設定的日期清單
3. 當日排程只發一封，管理員補完菜單後當天不再發信
4. 管理員進入系統時，即時檢查後一個月是否有未設定菜單的上班日
5. 若有未設定，自動跳轉至菜單設定頁面，並顯示彈出視窗與頂部警示橫幅，列出未設定日期
6. 管理員補完菜單後，當日再進入系統不再提醒

## 例外情況
- 無

## 資料欄位
- 通知日期（notify_date）
- 未設定菜單起始日（missing_from）
- 未設定菜單結束日（missing_to）
- 通知狀態（status：success / fail）

## API 路徑
- GET /api/admin/notifications/menu-check
- POST /api/admin/notifications/menu-check/dismiss
