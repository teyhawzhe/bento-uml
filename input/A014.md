# A014 上班日行事曆設定

## 使用者角色
- 管理員

## 主要流程
1. 管理員進入行事曆設定頁（主 TAB）
2. 管理員可點擊「預產行事曆」按鈕，系統產生整年行事曆（預設週六、日為非上班日），執行前跳出確認提示，確認後整年覆蓋
3. 畫面顯示行事曆（唯讀狀態），可切換年／月，非上班日以紅色標註
4. 管理員按「編輯」按鈕，進入編輯模式
5. 管理員點擊日期直接切換上班日／非上班日狀態
6. 切換月份時若有未儲存變更，系統提醒「尚有修改未更新，請先按更新」
7. 管理員按「更新」按鈕，儲存當前月份的異動，回到唯讀狀態
8. 管理員可上傳 CSV，上傳後跳出 CSV 內容預覽，確認後才更新資料

## CSV 格式
- 每行格式：`yyyymmdd,y/n`
- `y` = 上班日，`n` = 非上班日

## 例外情況
- 預產行事曆前需二次確認，避免誤覆蓋
- 切換月份時若有未儲存變更，提醒使用者先按更新
- CSV 上傳後需預覽確認才生效

## 資料欄位
- 日期（date）
- 是否上班日（is_workday，boolean）

## API 路徑
- GET /api/admin/calendar?year={year}&month={month}
- PUT /api/admin/calendar
- POST /api/admin/calendar/generate（預產整年）
- POST /api/admin/calendar/import（CSV 上傳）
